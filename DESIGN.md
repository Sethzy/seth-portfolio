# Design System

## Direction

Dark, narrow, tactile portfolio UI with dense but readable sections. The surface should feel closer to a personal engineering dossier than a marketing landing page.

## Tokens

- Background: near-black `#090909`
- Surface: `#111111`
- Elevated surface: `#151515`
- Border: `rgba(255, 255, 255, 0.09)`
- Text: `#f4f4f4`
- Muted text: `#9a9a9a`
- Subtle text: `#6f6f6f`
- Accent: warm white `#ffffff`
- Radius: 18px large panels, 999px pills, 12px tiles

## Typography

Use system sans-serif with tight, unfussy hierarchy. Body copy should be 15-16px with calm line-height. Headings should be compact, not oversized.

## Components

- Floating top nav in a translucent pill
- Intro card with avatar, marquee role line, and metadata grid
- Contribution heatmap strip
- Rounded tech tiles with icon and label
- Project cards with muted media panel and pill tags
- Timeline entries with numbered rail and small date/location metadata
- Service/component catalog cards with restrained hover motion
